# litegraph.js（索引）

## 项目介绍

`litegraph.js` 是一个基于 JavaScript 的可视化节点图引擎，可在浏览器中构建和运行类似蓝图（Blueprint）的节点流程。

## 源项目

本项目以官方仓库为准：

- 源项目地址：https://github.com/jagenjo/litegraph.js
